Chrome Origin

Google Chrome is a cross-platform web browser developed by Google. It was first released in 2008 for Microsoft Windows, and was later ported to Linux, macOS, iOS, and Android where it is the default browser built into the OS.
Chrome was created first as an open source product. Many developers were allowed to access the code and contribute, if they saw fit to do so.

Chrome tricks and tips

Instant Search

It is possible to highlight words and drag and drop them onto the extra space beside your search tabs. Making chrome open a new page that Google searches the word.

Alternative scrolling

You can scroll down your screen by pressing shift and spacebar at the same time.

Save all tabs

Pressing Ctrl-Shift-T or Cmd-Shift-D will save all tabs currently open in a small folder that will appear at the bottom

Choose Your Default Search Engine

You can change your own default search enginge in your browser settings. Learn how to change yourbrowser settings.

Move a downloaded file to a specific place

Downloads can be placed in specific folders in your computer. You can then drag and drop the download directly to your desktop or to any folder.

Save everything you download in a cloud-based folder

you must install the desktop synchronization program for the storage service you use. Most of them, such as Google Drive, Dropbox and OneDrive, provide these utilities for all common operating systems. Once configured, you will have a folder on your local hard drive that will always sync with a folder in your cloud storage. In the "Advanced" section of the Chrome settings you can click on "Change" in "Downloads" and create a subfolder where all the files you download online will be.

Create custom keyboard shortcuts in Chrome

Go to the main menu in the upper right corner and within it to "More tools> Extensions". Once there, at the bottom, the Google browser enables the "Key combinations" option, where you can generate custom keyboard commands to quickly access all the extensions you want.
